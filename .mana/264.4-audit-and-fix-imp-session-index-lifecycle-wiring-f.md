Goal: determine whether imp currently indexes saved sessions during normal runtime lifecycle, and if not, define/implement the smallest correct ownership seam so `session_search` is reliable.

Current state:
- `imp/crates/imp-core/src/session_index.rs` defines the SQLite/FTS index API.
- `imp/crates/imp-core/src/tools/session_search.rs` reads the index DB and reports `No sessions indexed yet` when the DB is absent.
- Current repo search has not surfaced a production callsite that invokes `SessionIndex::index_session(...)` during normal CLI/TUI/session lifecycle, which likely explains real-world search failure even when many raw sessions exist.
- This issue is adjacent to, but distinct from, the path-topology mismatch captured elsewhere.

Steps:
1. Inspect CLI, TUI, and `imp_session` lifecycle code for any production indexing hook on session end, resume, compaction, or explicit commands.
2. Confirm whether indexing is currently unwired, partially wired, or only test-only.
3. If unwired, choose the smallest durable ownership seam for indexing (for example session shutdown, post-assistant turn persistence, or a dedicated recovery/reindex command) without re-deciding the entire storage topology.
4. Record the decision and either implement the minimal fix or write the worker-ready implementation brief, depending on what the parent storage-contract work decides.
5. Ensure this job stays tightly scoped to lifecycle reliability for the existing session index, not the broader path normalization spec.

Files:
- `imp/crates/imp-core/src/session_index.rs`
- `imp/crates/imp-core/src/tools/session_search.rs`
- `imp/crates/imp-core/src/imp_session.rs`
- `imp/crates/imp-cli/src/lib.rs`
- `imp/crates/imp-tui/src/app.rs`

In scope:
- session-index lifecycle wiring
- reliable local conversation recovery for `session_search`

Out of scope:
- broad storage-path redesign
- replacing the current index format

Do not:
- conflate missing indexing with missing raw sessions
- silently create background indexing behavior without deciding the right lifecycle seam
